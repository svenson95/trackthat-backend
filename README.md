# TrackThat Backend

Backend API for the TrackThat mobile app.

## Tech Stack

- Java
- Spring Boot
- Maven
- MongoDB (Cloud)
- REST API

## Hosting

Hosted on Fly.io.

## Requirements

- Java JDK
- Maven

## Development

To run this project locally:

```bash
mvnw.cmd spring-boot:run
```

## Environment Variables

```
GOOGLE_CLIENT_ID=your-google-client-id
JWT_SECRET=your-secret
```
